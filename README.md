# Suwako-Chan Cubic - SteamVR Mod

> *"I love it. Bullet hell game mechanics in three dimensions is awesome."* - Palmer Luckey

**Suwako-Chan Cubic (すわこちゃんcubic)**, also known as **Suwapyon2**, is a Touhou fan game originally created by **UTG Software** and released in **2012** for PC. It was later released as a VR version on **Oculus Share** for the **Oculus DK1 and DK2** devkits.  

We have modded the game to support **SteamVR** and modern VR features.

## Features
- SteamVR Support  
- 6 DOF Tracking  
- 3D Depth Slider  
- High Resolution Support  
- High FPS Support  
- Motion Controller Support  
- Control Rebinding Support  

## Install Instructions
1. Download **"Suwako-Chan.Cubic.SteamVR.Mod.7z"** from the releases section.  
2. Unzip the file.  
3. Launch **SteamVR**.  
4. Run **run.bat**.  

The batch file will automatically download the game from the cached version of the original UTG Software website on the **WayBack Machine** and apply our mod.  

Have fun!
